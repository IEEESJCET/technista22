COMPETITION

WEB ENTWICKLER

IEEE SB SJCET is providing you an oppurtunity to test your skills on Web Development through Web-Entwickler

The Topic for Web Entwickler will be declared half hour prior to the beginning of the competition. Further instructions will be mailed to the participants

The participant will be given 48 hours to develop the website. The final code should be uploaded to a repo named web-entwickler-technista on your github profile

RULES AND REGULATIONS
  • 1. Frameworks are allowed
  • 2. Prebuilt templates/themes are not allowed
  • 3. Multiple or single pages are allowed.
  • 4. CSS/ JS libraries are allowed (Bootstrap, Bulma etc.)
  • 5. Use CDN links for CSS frameworks
  • 6. CSS file should not be minified
  • 7. You could make custom illustrations or logos for the website
  • 8. The website should be responsive.


Date: 23.05.2020 to 26.05.2020
